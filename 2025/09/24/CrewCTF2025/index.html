1. 定义与符号说明

  • 有限域：F = GF(2192)，由192次不可约多项式m(x)定义
  • LCG状态：sn ∈ F，满足递推 sn + 1 = a ⋅ sn + b（a, b ∈ F为常数）
  • 输出值：vn = ⌊sn/2128⌋，即sn的高64位（192位状态右移128位）
  • 比特提取：vn的第i个比特记为vn(i) ∈ {0, 1}（0 ≤ i < 64）

2. 状态序列的线性递归性

定理1：sn满足F上的2阶线性递归
证明：
由递推关系：

sn + 1 − asn = b
sn + 2 − asn + 1 = b
两式相减得：
sn + 2 − (a+1)sn + 1 + asn = 0  在 F 中
即S = {sn}是2阶线性递归序列，特征多项式为x2 − (a+1)x + a。

3. 高位比特的线性相关性

定理2：对任意固定i，比特序列{vn(i)}满足GF(2)上的线性递归
证明：
将sn表示为二进制向量sn ∈ GF(2)192，其高位64位为：
vn = (sn[128],sn[129],...,sn[191])T
由定理1，sn + 2 = (a+1)sn + 1 + asn（系数在GF(2)中对应比特运算）。
该式对每个比特位成立，故vn的每个分量满足相同线性递归：
vn + 2(i) = c1vn + 1(i) + c0vn(i)  在 GF(2) 中
其中c0, c1为常数（由a决定）。

4. 矩阵构造与递推求解

定义：构造观测矩阵M ∈ GF(2)c × (64k)，其中：
- c = 194（状态位数+2），k = 5（窗口大小）
- 每行对应vi的64个比特：M[i] = (vi(0),vi(1),...,vi(63))
- 每列对应连续k个v的同一位：M[:,j] = (vj(t),vj + 1(t),...,vj + c − 1(t))（t = j mod  64）

定理3：M的左核空间包含递推系数向量
证明：
由定理2，存在系数向量p = (p0,p1,...,pk − 1)使：
$\sum_{t=0}^{k-1} p_t v_{n+t}^{(i)} = 0 \quad \forall n,i$
该式等价于p ⋅ M = 0，即p在M的左核中。

5. 预测公式推导

从左核基向量p可提取递推关系：
$v_n = \sum_{t=0}^{k-1} p_t v_{n-k+t} \quad \text{在} \ \text{GF}(2)^{64} \ \text{中}$
（按位异或运算，对应GF(2)加法）
